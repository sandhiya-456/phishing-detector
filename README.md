# 🔐 Phishing Website Detection System

## 📌 Overview

This project aims to detect phishing websites using machine learning techniques and web technologies. It analyzes website features to determine whether a URL is legitimate or malicious.

---

## 🎯 Objectives

* Identify phishing websites
* Improve online security
* Provide real-time detection

---

## 🛠 Technologies Used

* Python
* HTML
* CSS
* JavaScript

---

## ⚙️ Features

* URL-based phishing detection
* User-friendly interface
* Machine learning model for classification

---

## 🚀 How to Run

1. Clone the repository
2. Install required Python libraries
3. Run the backend model
4. Open the frontend (HTML file)
